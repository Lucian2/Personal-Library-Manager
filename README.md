Personal Library Manager

This is a Personal Library Manager application that allows you to:

- View books
- Add new books
- Edit existing books
- Delete books
- Filter books by genre
- The application uses a mock Node.js server to simulate backend operations

Technologies Used

- React: Frontend library for building the user interface
- Material UI: UI components library for styling
- SWR: For data fetching and caching
- Axios: HTTP client for data fetching
- Mock Node.js server: Backend simulation for managing books

Requirements

Make sure you have the following installed:

- Node.js (Compatible with React 18.x)
- npm (Node Package Manager)

Setup and Running the Application

Follow these steps to set up and run the application locally:

- Clone the repository
- Navigate into the project directory
- Install dependencies: run : npm install
- Run the application: run: npm start
- Access the app in your browser (http://localhost:3000)

Mock Backend Setup

The application uses a mock Node.js server to simulate backend functionality for managing the books.
To run the mock server:

- cd mock-server
- npm install
- npm start
